Дипломная работа Яндекс Практикума на курсе [Веб-разработчика](https://practicum.yandex.ru/web/) `переписанная` на React.

### Стек

`MERN` - MongoDB, Express, ReactJS, Node.js

`Effector` | `TypeScript` ...

### Ссылки на проект

[Gh-Page](./).
